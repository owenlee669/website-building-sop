## 工具选择原则

### 1. 先免费，后付费
- 新手先用免费工具
- 有收入后再升级
- 避免不必要的开支

### 2. 按需选择
- 明确自己的需求
- 不要贪多
- 够用就好

### 3. 可以合买
- 几个人分摊成本
- 注意账号安全
- 建议2-3人合买

## 工具使用技巧

### 建立工具库
- 收藏常用的工具网站
- 建立书签文件夹
- 定期整理更新

### 学习快捷键
- 每个工具都有快捷键
- 学会后效率翻倍
- 特别是VS Code

### 自动化工作流
- 用Zapier连接工具
- 自动化重复任务
- 节省时间
