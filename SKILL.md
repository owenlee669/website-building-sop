# 建站三字经 SOP

> 站之初，筛词库，怼布局
> 看收录，收录稳，上初链
> 看反馈，反馈佳，加页面
> 看词库，词库涨，上优链
> 看排名，排名升，调内容
> 卡细节，观其变，静等候
> 上首页，悟其道，周复始

## 阶段一：建站之初（筛词库 + 怼布局）

### 1.1 筛词库

**目标**：找到适合新站的低竞争关键词

**工具**：
- Semrush Keyword Magic Tool
- Ahrefs Keywords Explorer
- Google Trends

**筛选标准**：
| 指标 | 要求 | 说明 |
|------|------|------|
| KD (关键词难度) | < 30 | 新站有机会排名 |
| 搜索量 | > 100/月 | 有基本流量价值 |
| intitle结果 | < 1000 | 竞争页面少 |
| 趋势 | 近12月上升 | 新词新站机会 |
| 商业意图 | 中高 | 便于后期变现 |

**操作步骤**：
1. 在Semrush输入种子词，导出相关关键词列表
2. 按KD升序排列，筛选KD<30的词
3. 用Google Trends验证趋势（近期开始上升为佳）
4. Google搜索 `intitle:"关键词"` 查看标题竞争度
5. 将筛选出的词按主题归类，建立词库表格

### 1.2 怼布局

**目标**：将词库合理分配到网站各页面

**页面类型分配**：
- **首页**：核心大词（1-3个）
- **分类页**：中等竞争度词
- **内容页/工具页**：长尾词（每个页面聚焦1个主词）

**TDH优化**：
- **Title**：主关键词 + 修饰词（60字符内）
- **Description**：包含关键词的行动号召（160字符内）
- **Headings**：H1包含主词，H2-H3覆盖长尾变体

**内链结构**：
```
首页
├── 分类A
│   ├── 页面A1
│   └── 页面A2
├── 分类B
│   ├── 页面B1
│   └── 页面B2
└── ...
```

## 阶段二：看收录（收录稳，上初链）

### 2.1 提交收录

**上线后立即执行**：
1. 提交Google Search Console
2. 提交sitemap.xml
3. 在首页放置Google Analytics代码

### 2.2 监控收录

**观察指标**：
- GSC中「网页」→「已编入索引」数量
- `site:你的域名` 搜索结果数

**收录正常信号**：
- 首页24-48小时内被收录
- 内页7-14天内陆续收录
- 收录数量稳定增长

### 2.3 上初链（初步外链）

**触发条件**：收录稳定增长2周以上

**初链渠道**（免费/低成本）：
| 渠道 | 方法 | 数量 |
|------|------|------|
| 社交媒体 | Twitter/X、Reddit相关板块分享 | 5-10条 |
| 书签网站 | Delicious、Pocket等 | 3-5条 |
| 目录提交 | DMOZ替代品、行业目录 | 5-10条 |
| 论坛签名 | 相关行业论坛 | 3-5条 |
| 博客评论 | 行业博客有价值评论 | 10-20条 |

**注意**：初链不求质量，但求自然、相关、稳定

---

## 阶段三：看反馈（反馈佳，加页面）

### 3.1 观察正反馈

**正反馈信号**：
- GSC展示次数上升
- 点击数开始增长
- 部分词进入前100名
- 用户停留时间>30秒

### 3.2 加页面（扩大词库覆盖）

**策略**：
1. 在原词库基础上，挖掘更多长尾变体
2. 每个新页面聚焦1个长尾词
3. 保持内容质量，避免垃圾页面

**内容生产节奏**：
- 初期：每周2-5页
- 稳定期：每周5-10页
- 爆发期：根据资源决定

---

## 阶段四：看词库（词库涨，上优链）

### 4.1 监控词库增长

**工具**：
- Semrush → Organic Research → Positions
- Ahrefs → Organic Keywords

**健康指标**：
- 每周新增排名关键词 > 10个
- 排名前20的词稳步增加
- 总排名词库持续增长

### 4.2 上优链（优质外链）

**触发条件**：词库连续增长1个月以上

**优质外链渠道**：
| 渠道 | DA要求 | 获取方式 |
|------|--------|----------|
| 客座博客 | DA>30 | 主动联系站长 |
| 资源页链接 | DA>40 | 提供有价值资源 |
| 行业目录 | DA>35 | 高质量付费目录 |
| 新闻稿发布 | DA>50 | PRWeb等付费渠道 |
| 断链建设 | DA>30 | 用Ahrefs找断链 |

**首页外链策略**：
- 每月2-5条高质量外链到首页
- 锚文本多样化（品牌词、裸链、部分匹配）
- 避免过度优化（精确匹配锚文本<20%）

---

## 阶段五：看排名（排名升，调内容）

### 5.1 关注前20名词库

**重点监控**：
- GSC中排名11-20的词（最容易进首页）
- 排名波动大的词
- 有搜索量但排名靠后的词

### 5.2 On-Page SEO优化

**分析首页前5名**：
1. 查看他们的Title/Description写法
2. 分析内容结构和长度
3. 检查内链布局
4. 观察页面加载速度
5. 研究用户体验设计

**优化动作清单**：
- [ ] 内容长度达到或超过前5名平均
- [ ] 增加相关LSI关键词
- [ ] 优化图片alt标签
- [ ] 添加FAQ部分（获取富媒体摘要）
- [ ] 提升页面加载速度（<3秒）
- [ ] 优化移动端体验
- [ ] 添加内部链接（从已有高权重页面）

---

## 阶段六：卡细节（观其变，静等候）

### 6.1 调整后的观察期

**关键原则**：调整后不要频繁改动

**观察周期**：
- 小调整：1-2周
- 大改版：3-4周
- 算法更新后：4-6周

### 6.2 排名波动正常心态

**正常波动**：
- 单日排名±5位：正常
- 单周排名±10位：正常
- 算法更新后波动：正常

**何时担心**：
- 连续2周持续下降
- 排名消失（被惩罚信号）
- 流量断崖式下跌

## 阶段七：上首页（悟其道，周复始）

### 7.1 复盘成功页面

**分析维度**：
1. 关键词选择逻辑
2. 内容创作方法
3. 外链建设策略
4. 技术优化细节
5. 时间节点把控

### 7.2 提炼可复用模式

**形成SOP**：
- 成功页面类型模板
- 高效外链渠道清单
- 内容创作检查清单
- 技术优化标准流程

### 7.3 批量复制

**周复始**：
1. 用成功模式选下一批词
2. 快速建站/加页面
3. 执行已验证的外链策略
4. 持续监控和优化

## 常见错误避坑

1. **急于求成** - 新站不要一上来就猛发外链，容易被惩罚
2. **忽视收录** - 没收录就谈排名是空中楼阁
3. **关键词过宽** - 新站不要碰KD>50的词
4. **内容单薄** - 每页至少500字，工具站要有实际功能
5. **外链单一** - 多样化外链来源，避免被算法打击
6. **频繁改版** - 调整后要给Google时间反应
7. **忽视移动端** - 60%+流量来自移动设备

## 进阶技巧

### 快速收录技巧
- 在高权重平台（Reddit、Medium）发布带链接的内容
- 使用Google Indexing API（需申请）
- 提交到新闻网站获取快速索引

### 外链建设心法
- 先给价值，再要链接
- 内容为王，链接自然来
- 质量 > 数量（长期）
- 相关性 > DA值

### 内容生产加速
- 用AI生成初稿，人工润色
- 建立内容模板库
- 外包低价值内容创作
- 批量生产同类页面
