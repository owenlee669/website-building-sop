## 快速导航

### 🎯 找词实战指南 (keyword-research-guide.md)

**核心内容**：
- 新词新站核心逻辑
- 5 大找词渠道详解（Google Trends / Semrush / Similarweb / Reddit / Vercel）
- 关键词分类标准（核心词/中等词/长尾词/微长尾）
- 词库管理模板
- 判断词是否值得做的 checklist

**关键要点**：
```
新词 = 过去12个月内出现的关键词
判断标准：
- KD < 30
- 搜索量 > 100/月
- intitle 结果 < 1000
- 趋势上升或稳定
```

**使用时机**：
- 阶段一：筛词建站
- 阶段三：加页面（扩展词库）
- 阶段七：周复始（选下一批词）

### 🔗 外链建设指南 (backlink-building-guide.md)

**核心内容**：
- 外链类型分级（S/A/B/C/D 五级）
- 初链建设策略（免费渠道清单）
- 优链建设策略（客座博客/资源页/断链建设/HARO）
- 外链避坑指南
- 锚文本策略
- 外链监控工具

**关键要点**：
```
初链期（1-2月）：50-100条，免费渠道为主
增长期（3-6月）：每月10-20条，客座博客+资源页
稳定期（6月+）：每月5-10条，高质量外链

锚文本比例：
- 品牌词 40%
- 裸链 30%
- 部分匹配 20%
- 精确匹配 10%
```

**使用时机**：
- 阶段二：收录后上初链
- 阶段四：词库涨后上优链

### 📝 On-Page SEO 指南 (onpage-seo-guide.md)

**核心内容**：
- 页面质量检查清单（8 大维度）
- 内容质量优化（E-E-A-T 原则）
- 元标签优化（Title/Description）
- 标题层级（H1-H6）
- 链接策略（内链/外链）
- 图片优化
- 精选摘要优化
- 结构化数据（Schema Markup）
- Core Web Vitals 优化
- 移动端优化

**关键要点**：
```
Title：50-60字符，核心关键词前置
Description：150-160字符，包含行动号召
H1：每页仅1个，必须包含主关键词
内容长度：首页1000-2000字，内容页800-2000字
图片：WebP格式，<100KB，懒加载

Core Web Vitals：
- LCP < 2.5s
- FID < 100ms
- CLS < 0.1
```

**使用时机**：
- 阶段一：怼布局（建站时）
- 阶段五：调内容（排名优化时）
