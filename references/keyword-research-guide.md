# 找词实战指南

## 新词新站核心逻辑

新词 = 过去12个月内出现的关键词（尤其是AI相关）

为什么新词适合新站？
1. 老站还没有针对这些词优化
2. 竞争页面少，intitle结果<1000
3. Google算法对新内容友好
4. 先发优势明显

**12字秘诀应用**：
- **词找词**：用种子词找到更多相关词
- **词找站**：搜索关键词，分析前10名站点
- **站找词**：用Ahrefs查看竞品排名的所有词
- **站找站**：用Similarweb找到更多竞争对手

---

## 找词渠道详解

### 1. Google Trends

**用法**：
- 输入种子词，查看「相关查询」
- 筛选「上升」标签
- 关注"Breakout"关键词

**实战技巧**：
- 每天刷一遍相关行业的趋势
- 设置邮件提醒
- 结合新闻热点找词
- **查看URL参数**：在Google Trends页面底部，查看相关查询的URL参数，可以发现更多词根

**高阶用法**：
```
1. 输入词根（如：ai, calculator, generator）
2. 查看「相关查询」→「上升」
3. 记录"Breakout"关键词（搜索量激增）
4. 将这些词放入Semrush进一步分析
```

---

### 2. Semrush Keyword Magic Tool

**筛选设置**：
- KD: 0-30
- Volume: 100+
- Word count: 2-5个词（长尾词）
- **排除品牌词**：Filter → Exclude → 输入已知品牌

**导出后处理**：
1. 按主题归类
2. 标记难度等级
3. 标注商业意图
4. **计算KDROI**：(Volume × CPC) / KD

**KDROI公式**（关键词投资回报率）：
```
KDROI = (Volume × CPC) / KD
```
- Volume：月均搜索量
- CPC：平均单次点击成本（反映商业价值）
- KD：关键词难度

**选词优先级**：
- ✅ 优先选择 KD < 29 且具有较高 CPC 潜力的词
- ⚠️ 避开季节词（流量不稳定）
- ❌ 避开品牌词（用户意图已锁定）

---

### 3. Similarweb竞品分析

**步骤**：
1. 找到竞品网站
2. 查看「搜索」→「关键词」
3. 筛选低KD词
4. 复制到自己词库

**进阶技巧**：
- 查看竞品的「流量来源」→找到他们获取流量的渠道
- 分析「自然落地页」→学习他们的页面布局
- 对比多个竞品→找到共同遗漏的机会词

---

### 4. Reddit/Twitter/HackerNews

**监控方法**：
- 订阅相关subreddit
- 关注行业KOL
- 设置关键词提醒

**Reddit找词技巧**：
```
1. 进入相关subreddit（如r/webdev, r/SEO）
2. 查看「Top」→「This Week/Month」
3. 关注高频出现的词汇
4. 查看评论区的用户需求表达
```

**Twitter找词技巧**：
- 关注行业大V的推文
- 查看 trending topics
- 搜索 "how to" + 你的领域

### 5. Vercel子域名发现

**原理**：
很多新站先用Vercel子域名测试，可以通过搜索找到新上线的网站，反推他们做的关键词。

**搜索语法**：
```
site:vercel.app "你的行业关键词"
site:netlify.app "你的行业关键词"
site:github.io "你的行业关键词"
```

**使用方法**：
1. 搜索上述语法
2. 查看最近上线的网站
3. 分析他们的关键词策略
4. 找到他们遗漏的机会词

### 6. AI导航站找词

**原理**：
AI导航站收录了大量新上线的AI工具，是发现新词的重要渠道。

**常用导航站**：
- Toolify.ai
- theresanaiforthat.com
- futurepedia.io
- futuretools.io

**使用方法**：
1. 浏览最新收录的工具
2. 查看工具名称和描述
3. 提取关键词放入Google Trends验证
4. 分析这些工具的功能定位

### 7. 谷歌自动补全

**方法**：
```
1. 在Google搜索框输入种子词
2. 记录自动补全建议
3. 在种子词前加"how to"、"what is"、"best"等前缀
4. 在种子词后加空格，看字母A-Z的补全建议
```

**示例**：
```
输入：ai image
补全：ai image generator, ai image upscaler, ai image editor...

输入：best ai
补全：best ai image generator, best ai writing tool...
```

## 关键词可行性判断（五步分析法）

> 来源：哥飞小课堂 - 《再次系统的聊聊如何判断一个关键词是否能做》

### 第一步：分析搜索结果类型

**目标**：判断首页和内页的比例

**操作**：
1. 谷歌搜索目标关键词
2. 统计前10名结果中首页和内页数量

**判断标准**：
| 情况 | 难度 | 建议 |
|------|------|------|
| 前10中8+个内页 | ⭐⭐ 低 | 新站有机会 |
| 前10中5-7个内页 | ⭐⭐⭐ 中 | 需要外链支持 |
| 前10中8+个首页 | ⭐⭐⭐⭐⭐ 高 | 不建议新站做 |

---

### 第二步：流量分析

**工具**：Similarweb / AITDK.com插件

**判断标准**：
| 月访问量 | 价值 | 建议 |
|---------|------|------|
| <1万 | 低 | 不值得单独做站 |
| 1-10万 | 中 | 可以作为内页 |
| 10-100万 | 高 | 值得投入 |
| >100万 | 极高 | 重点投入 |

---

### 第三步：外链分析

**工具**：Ahrefs Backlink Checker

**Ahrefs KD解读**：
| KD值 | 需要外链域名 | 难度 |
|------|------------|------|
| 0-10 | 0-10个 | ⭐ 简单 |
| 11-30 | 10-50个 | ⭐⭐ 中等 |
| 31-50 | 50-100个 | ⭐⭐⭐ 困难 |
| 51-70 | 100-200个 | ⭐⭐⭐⭐ 很难 |
| 71-100 | 200+个 | ⭐⭐⭐⭐⭐ 极难 |

**注意**：进前3需要的外链数通常是前10的2-3倍

---

### 第四步：趋势验证

**工具**：Google Trends

**判断标准**：
| 趋势类型 | 特征 | 建议 |
|---------|------|------|
| 新词上升 | 2023年后开始有搜索量 | ✅ 可以做 |
| 稳定增长 | 持续2-3年增长 | ✅ 重点做 |
| 季节性 | 每年固定时间高峰 | ⚠️ 注意timing |
| 下降趋势 | 峰值已过 | ❌ 不建议 |
| 突发热点 | 新闻事件驱动 | ⚠️ 短期流量 |

---

### 第五步：用户需求理解

**方法**：通过搜索结果理解

**操作**：
1. 打开谷歌搜索结果前3名
2. 分析页面提供的功能/内容
3. 总结用户真正想要什么

**关键问题**：
- 用户搜索这个词是为了解决什么问题？
- 现有结果是否完全满足了需求？
- 你能提供什么差异化价值？

## 词库管理模板

```
| 关键词 | KD | 搜索量 | CPC | KDROI | 难度 | 优先级 | 分配页面 | 状态 |
|--------|-----|--------|-----|-------|------|--------|----------|------|
| xxx | 15 | 320 | $2.5 | 53.3 | 中 | P1 | 首页 | 待建站 |
```

**字段说明**：
- **KDROI**：(搜索量 × CPC) / KD，数值越高越值得做
- **难度**：综合KD、竞争度、内容难度的主观评估
- **优先级**：P1（最高）、P2（高）、P3（中）、P4（低）
- **状态**：待建站/已建站/已优化/已放弃

## 判断词是否值得做的checklist

- [ ] KD < 30
- [ ] 搜索量 > 100/月
- [ ] intitle结果 < 1000
- [ ] 趋势上升或稳定
- [ ] 有商业变现潜力
- [ ] 能在3小时内完成页面
- [ ] 有至少5个相关长尾词可扩展
- [ ] 前10名中内页数量 > 5个
- [ ] 不是明显的品牌词
- [ ] 不是季节性过强的词

**评分标准**：
- 满足8条以上 → ⭐⭐⭐⭐⭐ 强烈推荐
- 满足6-7条 → ⭐⭐⭐⭐ 值得做
- 满足4-5条 → ⭐⭐⭐ 可以考虑
- 满足<4条 → ⭐⭐ 不建议

## 漫游找词SOP（每日工作）

> 来源：哥飞社群 - 《漫游找词SOP》

**步骤1：打开数据平台**
- ahrefs
- similarweb
- semrush
- Google trends

**步骤2：进入关键词搜索页面，搜索任意单词**

可以从自己感兴趣、有所了解的领域触发：
- 具体的对象：cat(猫)、dog(狗)
- 具体的产品/品牌：poe2(流放之路2)、genshin(原神)、Disney(迪士尼)
- 一些分类：editor(编辑器)、table(表格)
- 等等...

**步骤3：设定结果筛选条件**
- 排除NSFW类的关键词
- KD < 30（40其实也可以看看）
- 搜索量从高到低排序
- CPC（如果搜索结果很多，也可以加一点CPC挑一挑）
- 如果有【搜索意图】，则把【导航】类型去掉

**步骤4：打开Google趋势**
- 在Google Trends上输入各种词根
- 看底部的相关查询
- 查看url中最后的参数

**步骤5：词海漫游**
> 有人说炒A股像读博（每天学习各种新概念），我觉得找词也差不多

- 了解每个词是什么意思
- 理解用户在什么情况下会来搜这个词
- 用户搜这个词是为了得到什么东西
- 可以批量导出，并用KDROI公式进一步筛选

**步骤6：进一步确定搜索量**
1. **谷歌趋势对标**：用GPTs做参照（每天搜索量约5K）
2. **谷歌Ads关键词规划工具**：查看准确搜索量

**步骤7：观察对手（判断优化难度）**
- 去谷歌搜索结果看前几名的站
- 如果全是内页，大站也不多 → 机会很大
- 如果主词在首页的多 → 需要掂量
- 通过 `allintitle:` 判断专门优化这个词的网站有多少

## 关键词评估指标

### KGR（Keyword Golden Ratio）

**公式**：
```
KGR = allintitle页面数量 ÷ 关键词月搜索量
```

**判断标准**：
| KGR值 | 竞争程度 | 建议 |
|-------|----------|------|
| < 0.25 | ⭐ 极低 | 新站首选 |
| 0.25-1.0 | ⭐⭐ 低 | 可以做 |
| 1.0-2.0 | ⭐⭐⭐ 中 | 需要外链 |
| > 2.0 | ⭐⭐⭐⭐⭐ 高 | 不建议 |

**使用方法**：
1. Google搜索 `allintitle:"你的关键词"`
2. 查看搜索结果数量
3. 除以关键词月搜索量
4. 根据KGR值判断是否值得做

---

### EKGR（Enhanced KGR）

**增强版KGR，考虑更多因素**

**公式**：
```
EKGR = (allintitle结果 × KD) ÷ (搜索量 × CPC)
```

**判断标准**：
| EKGR值 | 竞争程度 | 建议 |
|--------|----------|------|
| < 0.1 | ⭐ 极低 | 强烈推荐 |
| 0.1-0.5 | ⭐⭐ 低 | 推荐做 |
| 0.5-1.0 | ⭐⭐⭐ 中 | 可以考虑 |
| > 1.0 | ⭐⭐⭐⭐⭐ 高 | 不建议 |

**优势**：
- 综合考虑KD和CPC
- 更适合商业词评估
- 更精准的判断标准

---

### KDROI（Keyword Difficulty ROI）

**关键词投资回报率**

**公式**：
```
KDROI = (Volume × CPC) ÷ KD
```

**判断标准**：
| KDROI值 | 价值评估 | 建议 |
|---------|----------|------|
| > 100 | ⭐⭐⭐⭐⭐ 极高 | 必须做 |
| 50-100 | ⭐⭐⭐⭐ 高 | 优先做 |
| 20-50 | ⭐⭐⭐ 中 | 可以做 |
| < 20 | ⭐⭐ 低 | 考虑放弃 |

**使用场景**：
- 多个词难以抉择时
- 评估词的商业价值
- 优先排序词库

## 常见问题

### Q1: 找不到KD<30的词怎么办？
**A**: 
1. 放宽到KD<40，但需要更多外链支持
2. 寻找更细分的长尾词
3. 关注新兴领域（如AI工具）
4. 用「词找词」扩展更多相关词

### Q2: 如何判断一个词是不是新词？
**A**:
1. Google Trends查看历史数据
2. 查看搜索结果页面的时间（是否有2023年前的结果）
3. 用Ahrefs查看该词的历史排名数据

### Q3: 找到的词已经被别人做了怎么办？
**A**:
1. 分析竞争对手的弱点（内容质量、用户体验、功能完整性）
2. 寻找差异化角度
3. 做更细分的版本
4. 如果对手很强，果断放弃，找下一个词

### Q4: 每天应该花多少时间找词？
**A**: 
- 新手：40%时间找词，20%时间上站，40%时间外链
- 有经验后：2小时找词 → 半小时上站
- 做一个站要两三天 → 花一周找词都不为过
