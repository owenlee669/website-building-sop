# 建站执行检查清单资源

本文件夹包含建站 SOP 各阶段的执行检查清单模板，帮助你系统性地完成每个阶段的任务。

## 文件说明

### checklist-template.md
完整的建站七阶段检查清单，涵盖：
- 阶段一：筛词建站（找词检查 + 建站检查）
- 阶段二：收录监控（每日检查 + 收录后动作）
- 阶段三：加页面（内容生产检查）
- 阶段四：上优链（外链建设 + 监控指标）
- 阶段五：调内容（排名监控 + On-Page 优化）
- 阶段六：静等候（观察期检查）
- 阶段七：周复始（复盘总结 + 规模化）

## 使用建议

1. **复制使用**：将 `checklist-template.md` 复制到你的项目文件夹中
2. **阶段执行**：每完成一个阶段的任务，勾选对应检查项
3. **进度追踪**：通过检查清单追踪建站进度，确保不遗漏关键步骤
4. **团队协作**：可与团队成员共享，统一执行标准

## 检查清单速览

### 阶段一：筛词建站

**找词检查**
- [ ] 用 Semrush 导出关键词列表
- [ ] 筛选 KD<30 的词
- [ ] 用 Google Trends 验证趋势
- [ ] 检查 intitle 竞争度
- [ ] 按主题归类词库

**建站检查**
- [ ] 确定网站类型（工具站/内容站）
- [ ] 购买域名（优先关键词域名）
- [ ] 搭建网站框架
- [ ] 完成首页 TDH 优化
- [ ] 创建首批内容页面（5-10 页）
- [ ] 设置 Google Analytics
- [ ] 提交 Google Search Console
- [ ] 提交 sitemap.xml

### 阶段二：收录监控

**每日检查**
- [ ] GSC 收录数量
- [ ] site:域名 搜索结果
- [ ] 页面索引状态

**收录后动作**
- [ ] 社交媒体分享（5-10 条）
- [ ] 书签网站提交（3-5 条）
- [ ] 目录提交（5-10 条）
- [ ] 论坛签名更新（3-5 条）

### 阶段三：加页面

**内容生产**
- [ ] 每周新增页面计划
- [ ] 每页 TDH 优化
- [ ] 内部链接建设
- [ ] 内容质量检查

### 阶段四：上优链

**外链建设**
- [ ] 客座博客目标清单
- [ ] 资源页链接目标
- [ ] 断链建设机会
- [ ] HARO 回复

**监控指标**
- [ ] 外链数量增长
- [ ] 外链质量评估
- [ ] 锚文本分布

### 阶段五：调内容

**排名监控**
- [ ] 前 20 名词库数量
- [ ] 排名波动记录
- [ ] 竞品页面分析

**On-Page 优化**
- [ ] 内容长度对比
- [ ] LSI 关键词添加
- [ ] Schema 标记添加
- [ ] 页面速度优化

### 阶段六：静等候

**观察期**
- [ ] 记录调整日期
- [ ] 每日排名检查
- [ ] 流量变化监控
- [ ] 避免频繁改动

### 阶段七：周复始

**复盘总结**
- [ ] 成功页面分析
- [ ] 失败页面总结
- [ ] 可复用模式提炼
- [ ] 下一批词选择

**规模化**
- [ ] 成功模式复制
- [ ] 批量建站计划
- [ ] 外包任务分配
- [ ] 效率工具应用

## 使用技巧

### 每日工作流
1. 打开检查清单
2. 查看当前阶段未完成的检查项
3. 优先处理高优先级任务
4. 完成后勾选对应项

### 周度复盘
1. 统计本周完成的检查项数量
2. 分析未完成任务的原因
3. 调整下周计划
4. 更新词库和外链清单

### 月度总结
1. 回顾各阶段完成情况
2. 提炼可复用的执行模式
3. 优化检查清单（根据实际经验）
4. 制定下月目标
